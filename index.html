Introduction

This is a dependency-free Vanilla JavaScript slider for vanillaJS's lover. It's lightweight (~7kb) and easy to use.

Its core workings are similar to Owl Carousel, but it doesn't require jquery. Currently it only provides some essential features of a slider and just support horizontal slider.

Installation

Include CSS file in your <head>

<link href="own-carousel.min.css" rel="stylesheet">

Include JS file in your <head>

<script src="own-carousel.min.js"></script>

HTML structure 

                    
<div class="own-carousel__container">
    <div class="own-carousel__outer">
        <div class="own-carousel">
            <div class="own-carousel__item">Your content</div>
            <div class="own-carousel__item">Your content</div>
            <div class="own-carousel__item">Your content</div>
            <div class="own-carousel__item">Your content</div>
            <div class="own-carousel__item">Your content</div>
            <div class="own-carousel__item">Your content</div>
            <div class="own-carousel__item">Your content</div>
            <div class="own-carousel__item">Your content</div>
        </div>
    </div>
    <div class="own-carousel__control">
        <button class="control__prev">«</button>
        <button class="control__next">»</button>
    </div>
</div>

<!-- You have to follow this HTML structure and class name -->
<!-- own-carousel__control is optional, but if you want to have a navigation for your carousel, follow this structure, add another class and custom if you want -->
<!-- You can add another class name in "own-carousel__container" to add margin, padding, width,... for your slider -->
<!-- You can also add another class name in "own-carousel__item" to custom your carousel items -->

And call the function in your main.js (or another name ðŸ˜†) after your <body> tag 

                    
document.addEventListener("DOMContentLoaded", () => {
    document.querySelector(".own-carousel__container").ownCarousel({
        itemPerRow:5, 
        itemWidth:19,
        responsive: {
            1000: [4, 24],
            800: [3, 33],
            600: [2, 49],
            400: [1, 100]
        },
    });
    responsive();
});

//If you have more than 2 carousel in your page, just add another class in your "own-carousel__container" div,
//like "own-1", "own-2",... for example, and change "own-carousel__container" to "own-1", "own-2",... in querySelector.
//Don't forget to add responsive() after ownCarousel(), it's necessary for determine which responsive option is suitable after loading the page, 
//if you don't need responsive and responsive property is an empty object, then you don't need to add responsive(). You only need to call this
//function once whether you have more than 1 carousel in your page.

Options

  • itemPerRow

    Type:

    Number

    Default:

    4

    Description:

    This library is designed for desktop-first, this is the number of items you want to see in largest screen.

  • itemWidth

    Type:

    Number

    Default:

    24

    Description:

    This is the width (in percent) of each carousel item.

    The gap between each item will be calculated automatically.

    For example: If your itemPerRow is 4 and itemWidth is 24, then the gap between each item will be (100-24*4)/3. Don't let your gap be negative!

  • loop

    Type:

    Boolean

    Default:

    true

    Description:

    Infinite loop your carousel, clone some items in the beginning and the end of your carousel to make loop effect.

  • responsive

    Type:

    Object

    Default:

    {}

    Description:

    This is a little bit complicated structure you have to follow.

                                    
responsive: {
    1000: [4, 24], //breakpoint <=1000px
    800: [3, 33], //breakpoint <=800px
    600: [2, 49], //breakpoint <=600px
    400: [1, 100] //breakpoint <=400px
}

    The value in array is itemPerRow and itemWidth of corresponding screen. In this case, the itemPerRow and itemWidth in ownCarousel() is for > 1000px screen.

  • draggable

    Type:

    Boolean

    Default:

    true

    Description:

    Touch and Mouse drag enabled.

  • mouseWheel

    Type:

    Boolean

    Default:

    false

    Description:

    Mouse wheel to move your carousel.

  • autoplay

    Type:

    Number

    Default:

    0

    Description:

    If the value is 0, auto play is off. If the value is a positive number, it will be the time (in millisecond) between 2 moves of your carousel.

    If you start dragging, autoplay will be paused. When you finish dragging, autoplay will be restarted in 2 seconds.

  • stopAutoplayWhenHover

    Type:

    Boolean

    Default:

    true

    Description:

    If autoplay is on, you may need this option. If the value is true (default), the autoplay will be paused when you hover on this carousel.

    When you finish hovering, autoplay will be restarted in 2 seconds.

  • nav

    Type:

    Boolean

    Default:

    false

    Description:

    If you used "own-carousel__control" in your HTML structure to add navigation, set nav's value to true to add functionality to your nav's button.
